Задание реализовано на Python
API REST реализовано с помощью библиотеки Flask
В связи с тем, что объем данных не большой, хранение данных осуществляется в глобальной переменной, тип переменной DataFrame
Скачивание и обработка данных производится с помощью библиотек request, BeautifulSoup, pandas

Основной файл - run.py, где происходит запуск сервера Flask(для API REST)
В этом же файле настраиваем api запросы.

Parse_api.py - Получает html по url (request), парсит полученный html(BeautifulSoup),
находит ссылку на скачивание данных в формате csv. Полученный файл скачивает, загружает (pandas).
С помощью библиотеки pandas производит манипуляции над данными - переименовываем колонки,
производим замену названия месяца на его обозначения в цифрах,
приводит данные в столбцах к определенному формату.

Фун-я get_info задает последовательность выполнения функций.

Проверка API
http://127.0.0.1:5000/price/2015-03-14
http://127.0.0.1:5000/mean_price/2015-03-04/2015-10-04
http://127.0.0.1:5000/min_max_price/2015-03-04/2015-10-04
http://127.0.0.1:5000/statistics/
